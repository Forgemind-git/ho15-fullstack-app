# HO15 Sample 2 — Booking App

## Your task

Your small business takes bookings over WhatsApp and double-books constantly. Build and deploy a booking / waitlist app.

## What you will build

A full CRUD app that runs in the browser using localStorage — no server or database needed.
Deploy free to GitHub Pages.

## Features to implement

**Fields:**
- Name
- Email
- Service
- Date
- Time

**Operations:**
- Submit booking
- View all bookings
- Check for conflicts (same date + time)
- Mark as Confirmed / Cancelled
- Send confirmation text (copy to clipboard)

## How to build it

1. Open index.html in your browser — you should see the skeleton layout
2. Read the TODO comments in the HTML and JavaScript
3. Fill in: the form fields, the localStorage functions, the list rendering
4. Test all operations (add, view, conflict detection, confirm/cancel)
5. Push to GitHub and enable Pages for a live link
